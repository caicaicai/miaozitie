# 🐱 喵字帖 - 小学生练字工具

一个专为小学生设计的在线汉字练字帖生成工具，支持笔画演示、描红练习和文章注音。

## 🌐 在线体验

[喵字帖](https://miaozitie.80wdb.com/) - Cloudflare Pages 部署版本

## ✨ 功能特点

### 🏠 导航首页 (`index.html`)
- 🗂️ **工具卡片导航**：一进站即可看到全部 6 个工具的卡片入口，点击即可进入
- 🎨 **专业简洁设计**：独立于各功能页面的品牌横幅 + 卡片网格布局

### 🎯 互动自测练字 (`zice.html`)
- ✍️ **在线书写判定**：基于 Hanzi Writer 笔顺引擎，鼠标/触屏按笔顺书写，实时判断对错
- 💡 **提示笔顺**：卡住时可查看完整笔顺动画演示，再继续书写
- 🎚️ **难度可调**：可开关提示轮廓、设置连续写错几次后自动提示
- 📊 **测验结果统计**：完成后按"完美/良好/需多练"分类展示，可只重测写错的字

### 📝 笔画练习模式 (`bihua.html`)
- 📚 **年级分类**：支持一到六年级（上下册）的生字库，共2500+个常用汉字
- 🖊️ **笔画演示**：逐笔显示汉字书写顺序，帮助学习正确笔顺
- ✍️ **描红练习**：提供田字格描红格和空白练习格，数量可分别设置
- 🔤 **拼音标注**：自动显示汉字拼音，学字认音同步进行
- 🎨 **自定义设置**：可调节格子大小、文字透明度、练习次数等
- 🖨️ **打印优化**：支持单列/双列布局，适配A4纸张

### ✍️ 抄字模式 (`chaozi.html`)
- 📐 **紧凑布局**：每个字占用极小空间，适合抄写文章
- 🔧 **灵活配置**：
  - 显示/隐藏拼音、原字、笔顺
  - 笔顺列数（3-12列）
  - 笔顺行数（1-6行）
  - 描红/空白练习格
- 📊 **多列打印**：支持 6列 / 8列 / 10列 布局
- 📏 **统一高度**：所有字的容器高度一致，布局整齐美观
- ➖ **分割线**：行列分割线，视觉更清晰

### 📖 文章注音 (`pinyin.html`)
- 📝 **批量转换**：输入任意中文文章，自动添加拼音注音
- 🎨 **样式定制**：可调节字体大小、拼音颜色、是否显示声调
- 📚 **预设故事**：内置多个经典故事示例
- 🖨️ **打印友好**：优化的打印样式，适合儿童阅读

### 🔊 生字听写 (`tingxie.html`)
- 🎲 **年级抽取 / 自定义内容**：可按年级随机抽取生字，也可自定义听写词语列表
- 🔊 **语音朗读**：调用浏览器语音朗读生字/词语，支持语速、朗读遍数设置
- ⏯️ **自动 / 手动播放**：自动定时播放下一个，或手动点击切换，适合不同节奏
- 📋 **答案核对 + 打印答题纸**：听写完成后可查看拼音答案，或打印田字格答题纸

### 🔤 看拼音写汉字 (`moxie.html`)
- 🎲 **年级抽取 / 自定义内容**：与生字听写一致的内容来源方式
- 📝 **默写练习纸**：每个字/词上方标注拼音，下方留出多个田字格供反复默写
- 🔁 **练习次数可调**：每个内容可设置 2~6 次重复练习
- 🖨️ **打印友好**：适配 A4 纸张，可直接打印使用

### 🏷️ 姓名贴生成器 (`mingtie.html`)
- ✏️ **一键批量**：输入班级、姓名、学号，自动生成整页姓名贴
- 🔢 **数量可调**：自定义打印数量，自动铺满 A4 页面
- 🖨️ **打印优化**：适配 A4 纸张，方便裁剪粘贴

### 🧮 口算速练 (`kousuan.html`)
- ➕ **多种运算**：支持加减乘除及混合运算，可自定义数值范围
- ⌨️ **数字键盘**：内置虚拟数字键盘，手机 / 平板均可流畅答题
- 🎯 **即时反馈**：自动判分、记录正确率，越算越熟练
- 🌙 **护眼模式 / 全屏模式**：适合长时间练习场景

### 📐 运算表打印 (`yunsuanbiao.html`)
- ✖️➕➖ **三种运算表**：乘法表（经典九九表样式）、加法表、减法表
- 🔢 **范围可调**：乘法表支持 1~6/9/10/12，加减法表支持 5~50 以内
- 🕳️ **填空练习模式**：可随机挖空部分格子，打印后当堂练习
- 🖨️ **一键打印**：适配 A4 纸张

### 🕐 时钟认读练习 (`shizhong.html`)
- 🎮 **互动答题**：随机显示模拟时钟，选择时/分作答，即时判断对错并统计正确率
- 🖨️ **打印练习卷**：批量生成多个时钟图案 + 空白作答线，适合课堂练习
- 🎯 **精度可调**：整时 / 5 分钟 / 任意分钟三档难度

## 🚀 快速开始

### 本地使用

直接在浏览器中打开以下任一文件即可使用：

| 文件 | 功能 | 适用场景 |
|------|------|----------|
| `index.html` | 导航首页 | 浏览并进入各个工具 |
| `bihua.html` | 笔画练习 | 详细学习笔顺，逐笔练习 |
| `zice.html` | 互动自测练字 | 在线书写自测，实时判断对错 |
| `chaozi.html` | 抄字模式 | 抄写文章句子，紧凑省纸 |
| `pinyin.html` | 文章注音 | 阅读故事，学习拼音 |
| `moxie.html` | 看拼音写汉字 | 看拼音默写汉字练习 |
| `tingxie.html` | 生字听写 | 语音朗读听写，检验生字掌握 |
| `mingtie.html` | 姓名贴生成器 | 批量制作班级姓名贴 |
| `kousuan.html` | 口算速练 | 手机/平板口算训练 |
| `yunsuanbiao.html` | 运算表打印 | 乘法/加法/减法表打印 |
| `shizhong.html` | 时钟认读练习 | 互动答题或打印练习卷 |

### 使用步骤

打开 `index.html` 首页，点击工具卡片即可进入对应功能；每个功能页顶部都有"← 返回首页"链接，方便随时切换。

#### 笔画练习
1. 选择年级（可选）
2. 输入汉字或点击"随机生成"
3. 调整描格/练格数量、透明度等参数
4. 点击"打印练字帖"

#### 互动自测练字
1. 按年级随机抽取，或输入自定义汉字
2. 设置提示轮廓和自动提示时机
3. 点击"开始自测"，按笔顺在方格中书写
4. 完成后查看测验结果，可只重测写错的字

#### 抄字模式
1. 输入要抄写的文章或句子
2. 配置显示选项（拼音/原字/笔顺）
3. 选择练习格类型（描红/空白）
4. 调整打印列数
5. 点击"打印"

#### 文章注音
1. 输入或粘贴中文文章
2. 调整字体大小和拼音颜色
3. 点击"生成注音"
4. 点击"打印"

#### 生字听写
1. 按年级随机抽取，或输入自定义听写内容
2. 设置语速、朗读遍数、自动/手动播放方式
3. 点击"开始听写"，孩子边听边写
4. 听写完成后查看答案核对，或打印听写答题纸

#### 看拼音写汉字
1. 按年级随机抽取，或输入自定义字词
2. 设置每个内容的练习次数
3. 点击"生成练习纸"并打印

#### 姓名贴生成器
1. 输入班级、姓名、学号
2. 设置打印数量
3. 点击"生成姓名贴"并打印

#### 口算速练
1. 选择运算类型和数值范围
2. 点击开始练习，用虚拟数字键盘答题
3. 查看正确率统计，可开启护眼/全屏模式

#### 运算表打印
1. 选择乘法表/加法表/减法表及范围
2. 可选开启填空练习模式
3. 点击"生成运算表"并打印

#### 时钟认读练习
1. 互动练习：设置时间精度和题量，点击"开始练习"，选择时/分作答
2. 打印练习卷：设置数量和精度，点击"生成练习卷"并打印

```
miaozitie/
├── index.html          # 导航首页（工具卡片入口）
├── bihua.html          # 笔画练习
├── zice.html           # 互动自测练字
├── chaozi.html         # 抄字模式
├── pinyin.html         # 文章注音
├── moxie.html          # 看拼音写汉字
├── tingxie.html        # 生字听写
├── mingtie.html        # 姓名贴生成器
├── kousuan.html        # 口算速练
├── yunsuanbiao.html    # 运算表打印
├── shizhong.html       # 时钟认读练习
├── common.css          # 全站共享样式（页头/导航/页脚）
├── ziku-data.js        # 共享生字库数据（bihua.html / tingxie.html / zice.html / moxie.html 复用）
├── hanzi-writer.min.js # 汉字笔画库
├── pinyin-pro.min.js   # 拼音转换库
├── tf.min.js           # 口算速练依赖库
├── ziku.txt            # 小学生字库（文档参考）
├── README.md           # 项目说明
└── DEPLOYMENT.md       # 部署指南
```
