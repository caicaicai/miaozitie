# 喵字帖 - 小学生笔画练字帖生成器

一个专为小学生设计的在线汉字练字帖生成工具，支持笔画演示和描红练习。

## ✨ 功能特点

- 📚 **年级分类**：支持一到六年级（上下册）的生字库，共2500+个常用汉字
- 🖊️ **笔画演示**：逐笔显示汉字书写顺序，帮助学习正确笔顺
- ✍️ **描红练习**：提供田字格练习格，支持自定义练习次数
- 🔤 **拼音标注**：自动显示汉字拼音，学字认音同步进行
- 🎨 **自定义设置**：可调节格子大小、文字透明度、练习次数等
- 🖨️ **打印优化**：双列布局，适配A4纸张，节省纸张

## 🚀 快速开始

- 直接在浏览器中打开 `stroke-by-stroke.html` 即可使用。
- [喵字帖](https://miaozitie.80wdb.com/) - cloudflare部署版本


### 使用步骤

1. **选择年级**：从下拉菜单选择对应年级
2. **生成练字帖**：
   - 手动输入：在文本框输入要练习的汉字
   - 随机生成：点击"随机生成"按钮从所选年级随机抽取汉字
3. **调整设置**：根据需要调整格子大小、透明度、练习次数等
4. **打印**：点击"打印练字帖"按钮，预览并打印
